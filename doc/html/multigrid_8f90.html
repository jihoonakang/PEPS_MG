module  multigrid
 Module for solving Poisson equation with V-cycle multigrid(MG) method. CGS, CGA and CGPSA are supported in this module. CGS : Coarse grid solver without grid aggregation. Detailed Description

This file contains a module for V-cycle multigrid method with CGS, CGA, and CGPSA CGS : Coarse grid solver without grid aggregation. CGA : Coarse grid aggregation in a target level. All grid variables are aggregated into a singel core. CGPSA : Coarse grid partial semi-aggregation in multiple levels.
