Detailed Description

Partitioned subdomain information.

It contains the geometry of subdomain in three-dimensional Cartesian cooridnates. Dimensions of grids and meshes are included. Cell-centered grid system is employed. Grid variables, solution, RHS and residual, is defined in this type variable.
